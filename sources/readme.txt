Necessary DLLs
==============

1. 3rdparty/dll/libgcc_s_dw2-1.dll
2. 3rdparty/dll/glut32.dll
3. bin/release/engine.dll

Enumerated libraries should be copied together with output project (.exe file)



Linux building
==============
1. Install free-glut, free-glut devel using package manager
2. Install CodeBlocks
3. Just update from SVN and write code!



Windows building
================
1. Install CodeBlocks
2. Just update from SVN and write code!



Filename conventions
====================
1. Textures: name_{T}.{EXT}; T = {d (diffuse), n (normals), s (specular), o (opacity)}, EXT = {tga, vet}
2. Objects: name_{RACE}_{LEVEL}.veo
